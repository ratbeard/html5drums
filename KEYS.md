Right hand is generally for moving around
Left hand is generally for selecting modes


Default Mode
============

`j`  right
`⇧j` left
`k`  down
`⇧k` up


Column Mode
===========
Enter with `f` from Default Mode
give coordinates


Row Mode
========
Enter with `d` from Default Mode
give coordinates


Beat Mode
=========
Enter with `t` from Default Mode
map keys to sounds
